# Тест-кейс на позитивную проверку
ID1
Тест-кейс на позитивную проверку

Шаги: 
1. Перейти на сайт https://www.saucedemo.com/
2. Ввести логин в ячейку Username (standard_user/problem_user/performance_glitch_user/error_user/visual_user)
3. Ввести пароль в ячейку Password (secret_sauce)
4. Нажать кнопку Login  

Ожидаемый результат: 
1. Отображается форма авторизации с полями Username, Password и кнопкой Login
2. Логин отображается в поле ввода
3. Пароль отображается в замаскированном виде (точки)
4. Переход на страницу Swag Labs
